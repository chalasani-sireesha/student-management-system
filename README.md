# Student Management System (Django)

A simple Django-based Student Management System with CRUD operations, search, pagination, and Bootstrap UI.

---

## 🚀 Features

- Add Student
- View Students
- Update Student
- Delete Student
- Search Students
- Pagination
- Success Messages
- Bootstrap UI Design

---

## 🛠️ Tech Stack

- Python
- Django
- SQLite
- HTML
- Bootstrap
